# MX-HHKBLE
Using the [nice!nano](https://docs.nicekeyboards.com/#/), making a low-profile MX-compatible HHKB.

# Key Switches
[Kailh low profile round switches](https://mechanicalkeyboards.com/shop/index.php?l=product_detail&p=6343)

    * Working on that now
